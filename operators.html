C Operators

Operators are symbols that tell the compiler to perform specific mathematical or logical operations.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, etc.

Operator Description Example
+ Addition a + b = 30
- Subtraction a - b = 10
* Multiplication a * b = 200
/ Division b / a = 2
% Modulus (remainder after division) b % a = 0
++ Increment (increases value by 1) a++ results in a = 21
-- Decrement (decreases value by 1) a-- results in a = 19

Example: Arithmetic Operators

/**
 * Demonstration of C Arithmetic Operators
 * This program shows the usage of all arithmetic operators in C
 */
#include <stdio.h>

int main() {
    // Initialize variables
    int a = 20, b = 10, c;
    
    // Addition
    c = a + b;
    printf("Addition: a + b = %d\n", c);  // 30
    
    // Subtraction
    c = a - b;
    printf("Subtraction: a - b = %d\n", c);  // 10
    
    // Multiplication
    c = a * b;
    printf("Multiplication: a * b = %d\n", c);  // 200
    
    // Division
    c = a / b;
    printf("Division: a / b = %d\n", c);  // 2
    
    // Modulus (remainder)
    c = a % b;
    printf("Modulus: a %% b = %d\n", c);  // 0
    
    // Division with decimal result
    float x = 15.0, y = 4.0, z;
    z = x / y;
    printf("Float division: %.1f / %.1f = %.2f\n", x, y, z);  // 3.75
    
    // Increment operators
    printf("\nIncrement/Decrement Operators:\n");
    printf("Original values: a = %d, b = %d\n", a, b);  // a = 20, b = 10
    
    // Postfix increment (use then increment)
    printf("Postfix a++: %d\n", a++);  // 20
    printf("After a++: a = %d\n", a);   // 21
    
    // Prefix decrement (decrement then use)
    printf("Prefix --b: %d\n", --b);    // 9
    printf("After --b: b = %d\n", b);   // 9
    
    // Prefix increment (increment then use)
    a = 20;  // Reset a
    printf("\nPrefix ++a: %d\n", ++a);  // 21
    printf("After ++a: a = %d\n", a);   // 21
    
    // Postfix decrement (use then decrement)
    b = 10;  // Reset b
    printf("Postfix b--: %d\n", b--);  // 10
    printf("After b--: b = %d\n", b);   // 9
    
    return 0;
}

Output:


Addition: a + b = 30
Subtraction: a - b = 10
Multiplication: a * b = 200
Division: a / b = 2
Modulus: a % b = 0
Float division: 15.0 / 4.0 = 3.75

Increment/Decrement Operators:
Original values: a = 20, b = 10
Postfix a++: 20
After a++: a = 21
Prefix --b: 9
After --b: b = 9

Prefix ++a: 21
After ++a: a = 21
Postfix b--: 10
After b--: b = 9

Note: The increment (++) and decrement (--) operators can be used as prefix or postfix. When used as prefix (++a), the value is incremented first and then used. When used as postfix (a++), the current value is used first and then incremented.

Relational Operators

Relational operators are used to compare two values.

Operator Description Example
== Equal to (a == b) is false
!= Not equal to (a != b) is true
> Greater than (a > b) is true
< Less than (a < b) is false
>= Greater than or equal to (a >= b) is true
<= Less than or equal to (a <= b) is false

Example: Relational Operators

/**
 * Demonstration of C Relational Operators
 * This program shows how relational operators compare values
 * and return boolean results (1 for true, 0 for false)
 */
#include <stdio.h>
#include <stdbool.h>  // For bool type in C99 and later

int main() {
    // Initialize variables
    int a = 20, b = 10, c = 20;
    
    // Print header
    printf("Relational Operators Comparison (1=true, 0=false)\n");
    printf("-------------------------------------------------\n");
    printf("Values: a = %d, b = %d, c = %d\n\n", a, b, c);
    
    // Equal to
    printf("Equal to (==):\n");
    printf("a == b: %d\n", a == b);     // 0 (false)
    printf("a == c: %d\n\n", a == c);   // 1 (true)
    
    // Not equal to
    printf("Not equal to (!=):\n");
    printf("a != b: %d\n", a != b);     // 1 (true)
    printf("a != c: %d\n\n", a != c);   // 0 (false)
    
    // Greater than
    printf("Greater than (>):\n");
    printf("a > b: %d\n", a > b);       // 1 (true)
    printf("a > c: %d\n\n", a > c);     // 0 (false)
    
    // Less than
    printf("Less than (<):\n");
    printf("a < b: %d\n", a < b);       // 0 (false)
    printf("b < a: %d\n\n", b < a);     // 1 (true)
    
    // Greater than or equal to
    printf("Greater than or equal to (>=):\n");
    printf("a >= b: %d\n", a >= b);     // 1 (true)
    printf("a >= c: %d\n\n", a >= c);   // 1 (true)
    
    // Less than or equal to
    printf("Less than or equal to (<=):\n");
    printf("a <= b: %d\n", a <= b);     // 0 (false)
    printf("a <= c: %d\n", a <= c);     // 1 (true)
    
    // Using relational operators in if statements
    printf("\nUsing relational operators in if statements:\n");
    if (a > b) {
        printf("a is greater than b\n");
    }
    
    if (a == c) {
        printf("a is equal to c\n");
    }
    
    return 0;
}

Output:

-------------------------------------------------
Values: a = 20, b = 10, c = 20

Equal to (==):
a == b: 0
a == c: 1

Not equal to (!=):
a != b: 1
a != c: 0

Greater than (>):
a > b: 1
a > c: 0

Less than (<):
a < b: 0
b < a: 1

Greater than or equal to (>=):
a >= b: 1
a >= c: 1

a <= b: 0
a <= c: 1

Using relational operators in if statements:
a is greater than b
a is equal to c

Note: In C, relational operators return 1 for true and 0 for false.

Logical Operators

Logical operators are used to combine conditional statements.

Operator Description Example
&& Logical AND (a > b) && (a != b) is true
|| Logical OR (a < b) || (a != b) is true
! Logical NOT !(a == b) is true

Example: Logical Operators

/**
 * Demonstration of C Logical Operators
 * This program shows how logical operators combine conditions
 * and demonstrates short-circuit evaluation
 */
#include <stdio.h>
#include <stdbool.h>  // For bool type in C99 and later

// Function to demonstrate short-circuit evaluation
bool checkCondition(const char* name, bool returnValue) {
    printf("Checking condition %s...\n", name);
    return returnValue;
}

int main() {
    // Initialize variables
    int a = 20, b = 10, c = 15;
    bool result;
    
    printf("Logical Operators (1=true, 0=false)\n");
    printf("----------------------------------\n");
    printf("Values: a = %d, b = %d, c = %d\n\n", a, b, c);
    
    // Logical AND (&&)
    printf("Logical AND (&&):\n");
    result = (a > b) && (a > c);  // true && true
    printf("(a > b) && (a > c): %d\n", result);  // 1 (true)
    
    result = (a < b) && (a > c);  // false && true
    printf("(a < b) && (a > c): %d\n\n", result);  // 0 (false)
    
    // Logical OR (||)
    printf("Logical OR (||):\n");
    result = (a > b) || (a < c);  // true || false
    printf("(a > b) || (a < c): %d\n", result);  // 1 (true)
    
    result = (a < b) || (a < c);  // false || false
    printf("(a < b) || (a < c): %d\n\n", result);  // 0 (false)
    
    // Logical NOT (!)
    printf("Logical NOT (!):\n");
    result = !(a > b);  // !true
    printf("!(a > b): %d\n", result);  // 0 (false)
    
    result = !(a < b);  // !false
    printf("!(a < b): %d\n\n", result);  // 1 (true)
    
    // Short-circuit evaluation
    printf("\nShort-circuit evaluation:\n");
    printf("\nLogical AND with short-circuit:\n");
    
    // First condition is false, second won't be evaluated
    if ((a < b) && checkCondition("second AND condition", true)) {
        printf("This won't be printed\n");
    } else {
        printf("AND short-circuit: Second condition was not evaluated\n");
    }
    
    printf("\nLogical OR with short-circuit:\n");
    
    // First condition is true, second won't be evaluated
    if ((a > b) || checkCondition("second OR condition", true)) {
        printf("OR short-circuit: Second condition was not evaluated\n");
    }
    
    // Using logical operators in complex conditions
    printf("\nComplex condition: (a > b) && (b < c) || (c != a)\n");
    result = (a > b) && (b < c) || (c != a);
    printf("Result: %d\n", result);  // 1 (true)
    
    return 0;
}

Output:


Logical Operators (1=true, 0=false)
----------------------------------
Values: a = 20, b = 10, c = 15

Logical AND (&&):
(a > b) && (a > c): 1
(a < b) && (a > c): 0

Logical OR (||):
(a > b) || (a < c): 1
(a < b) || (a < c): 0

Logical NOT (!):
!(a > b): 0
!(a < b): 1

Short-circuit evaluation:

AND short-circuit example:
Checking d = 0
Result: 0

OR short-circuit example:
Checking a = 20
Result: 1

Practical examples:
a is the largest of the three values
b is not the smallest value
Pointer is NULL or value is not 10

Note: The logical AND (&&) operator evaluates the second operand only if the first operand is true. The logical OR (||) operator evaluates the second operand only if the first operand is false. This is called short-circuit evaluation.

Bitwise Operators

Bitwise operators perform operations on individual bits of integer values.

Operator Description Example
& Bitwise AND a & b = 0 (0000 in binary)
| Bitwise OR a | b = 30 (11110 in binary)
^ Bitwise XOR (exclusive OR) a ^ b = 30 (11110 in binary)
~ Bitwise NOT (one's complement) ~a = -21 (in decimal, assuming 32-bit integer)
<< Left shift a << 2 = 80 (shifts bits left by 2 positions)
>> Right shift a >> 2 = 5 (shifts bits right by 2 positions)

Example: Bitwise Operators

/**
 * Demonstration of C Bitwise Operators
 * This program shows how bitwise operators manipulate individual bits
 * in integer values
 */
#include <stdio.h>

// Function to print binary representation of a number
void printBinary(unsigned int num) {
    unsigned int mask = 1 << 7; // Start with the leftmost bit (8-bit display)
    printf("Binary: ");
    
    // Print 8 bits for readability
    for(int i = 0; i < 8; i++) {
        printf("%d", (num & mask) ? 1 : 0);
        
        // Add space every 4 bits for readability
        if(i == 3) {
            printf(" ");
        }
        
        mask >>= 1; // Shift the mask to the right
    }
    printf("\n");
}

int main() {
    // Initialize variables
    unsigned int a = 60;  /* 60 = 0011 1100 in binary */
    unsigned int b = 13;  /* 13 = 0000 1101 in binary */
    unsigned int c;
    
    printf("Bitwise Operators Demonstration\n");
    printf("--------------------------------\n\n");
    
    printf("Values:\n");
    printf("a = %u (decimal)\n", a);
    printBinary(a);       // 0011 1100
    printf("b = %u (decimal)\n", b);
    printBinary(b);       // 0000 1101
    printf("\n");
    
    // Bitwise AND
    c = a & b;  /* 12 = 0000 1100 in binary */
    printf("Bitwise AND (a & b):\n");
    printf("Result = %u\n", c);
    printBinary(c);
    printf("Explanation: Each bit is 1 only if both corresponding bits are 1\n\n");
    
    // Bitwise OR
    c = a | b;  /* 61 = 0011 1101 in binary */
    printf("Bitwise OR (a | b):\n");
    printf("Result = %u\n", c);
    printBinary(c);
    printf("Explanation: Each bit is 1 if at least one corresponding bit is 1\n\n");
    
    // Bitwise XOR
    c = a ^ b;  /* 49 = 0011 0001 in binary */
    printf("Bitwise XOR (a ^ b):\n");
    printf("Result = %u\n", c);
    printBinary(c);
    printf("Explanation: Each bit is 1 only if exactly one corresponding bit is 1\n\n");
    
    // Bitwise NOT (complement)
    c = ~a;     /* Inverts all bits of a */
    printf("Bitwise NOT (~a):\n");
    printf("Result = %d\n", c);  // Signed result
    printBinary((unsigned char)c); // Show just the lowest 8 bits
    printf("Explanation: Inverts all bits (0 becomes 1, 1 becomes 0)\n\n");
    
    // Left Shift
    c = a << 2; /* 240 = 1111 0000 in binary */
    printf("Left Shift (a << 2):\n");
    printf("Result = %u\n", c);
    printBinary(c);
    printf("Explanation: Shifts all bits 2 positions to the left, filling with 0s\n\n");
    
    // Right Shift
    c = a >> 2; /* 15 = 0000 1111 in binary */
    printf("Right Shift (a >> 2):\n");
    printf("Result = %u\n", c);
    printBinary(c);
    printf("Explanation: Shifts all bits 2 positions to the right, filling with 0s\n\n");
    
    return 0;
}

Result = 49

Result = -61

Practical Examples:

Note: Bitwise operators are commonly used in low-level programming, device drivers, and when working with hardware interfaces. They are also useful for optimizing certain operations and for manipulating individual bits in data.

Assignment Operators

Assignment operators are used to assign values to variables.

Operator Description Example
= Simple assignment c = a + b
+= Add and assign c += a is equivalent to c = c + a
-= Subtract and assign c -= a is equivalent to c = c - a
*= Multiply and assign c *= a is equivalent to c = c * a
/= Divide and assign c /= a is equivalent to c = c / a
%= Modulus and assign c %= a is equivalent to c = c % a
<<= Left shift and assign c <<= 2 is equivalent to c = c << 2
>>= Right shift and assign c >>= 2 is equivalent to c = c >> 2
&= Bitwise AND and assign c &= 2 is equivalent to c = c & 2
^= Bitwise XOR and assign c ^= 2 is equivalent to c = c ^ 2
|= Bitwise OR and assign c |= 2 is equivalent to c = c | 2

Example

#include <stdio.h>

int main() {
    int a = 20;
    int c = 100;
    
    // Simple assignment
    c = a;
    printf("c = a: c = %d\n", c);
    
    // Add AND assignment
    c += a;
    printf("c += a: c = %d\n", c);
    
    // Subtract AND assignment
    c -= a;
    printf("c -= a: c = %d\n", c);
    
    // Multiply AND assignment
    c *= a;
    printf("c *= a: c = %d\n", c);
    
    // Divide AND assignment
    c /= a;
    printf("c /= a: c = %d\n", c);
    
    // Modulus AND assignment
    c = 100;
    c %= a;
    printf("c %%= a: c = %d\n", c);
    
    // Bitwise AND assignment
    c = 100;
    c &= a;
    printf("c &= a: c = %d\n", c);
    
    // Bitwise OR assignment
    c = 100;
    c |= a;
    printf("c |= a: c = %d\n", c);
    
    // Bitwise XOR assignment
    c = 100;
    c ^= a;
    printf("c ^= a: c = %d\n", c);
    
    // Left shift assignment
    c = 100;
    c <<= 2;
    printf("c <<= 2: c = %d\n", c);
    
    // Right shift assignment
    c = 100;
    c >>= 2;
    printf("c >>= 2: c = %d\n", c);
    
    return 0;
}

Output:


c = a: c = 20
c += a: c = 40
c -= a: c = 20
c *= a: c = 400
c /= a: c = 20
c %= a: c = 0
c &= a: c = 4
c |= a: c = 116
c ^= a: c = 112
c <<= 2: c = 400
c >>= 2: c = 25

Note: Assignment operators provide a more concise way to update variables and are commonly used in loops and other situations where a variable's value needs to be modified repeatedly.

Miscellaneous Operators

C provides several other operators for various purposes.

Operator Description Example
sizeof Returns the size of a variable or data type in bytes sizeof(int) returns 4 (on most systems)
& Address operator (returns the memory address) &a returns the address of variable a
* Pointer operator (dereferences a pointer) *ptr returns the value at the address stored in ptr
?: Conditional (ternary) operator condition ? expr1 : expr2
, Comma operator (evaluates multiple expressions) expr1, expr2, expr3
. Direct member access operator struct.member
-> Indirect member access operator (via pointer) ptr->member

Example

#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

int main() {
    // Basic variable
    int a = 10;
    
    // Address operator (&) and pointer declaration
    int *ptr = &a;
    
    // sizeof operator
    int size = sizeof(a);
    
    // Conditional (ternary) operator
    int result = (a > 5) ? 100 : 200;
    
    // Structure and direct member access (.)
    struct Person person;
    person.age = 30;
    
    // Pointer to structure
    struct Person *personPtr = &person;
    
    // Using sizeof operator
    printf("Size of int: %d bytes\n", sizeof(int));
    
    // Using address operator
    printf("Address of a: %p\n", &a);
    
    // Using dereference operator
    printf("Value at ptr: %d\n", *ptr);
    
    // Result of ternary operator
    printf("Ternary result: %d\n", result);
    
    // Comma operator example
    int x = (a++, a+5, a*2);
    printf("x = %d, a = %d\n", x, a);
    
    // Direct member access
    printf("Person age: %d\n", person.age);
    
    // Indirect member access
    personPtr->age = 35;
    printf("Person age via pointer: %d\n", personPtr->age);
    
    return 0;
}

Output:


Size of int: 4 bytes
Address of a: 0x7ffd5c3e1a0c (this will vary)
Value at ptr: 10
Ternary result: 100
x = 22, a = 11
Person age: 30
Person age via pointer: 35

Note: The sizeof operator is evaluated at compile time, not runtime, unless the operand is a variable-length array.

Operator Precedence

Operator precedence determines the order in which operators are evaluated in an expression. Operators with higher precedence are evaluated before operators with lower precedence.

Category Operator Associativity
Postfix () [] -> . ++ -- Left to right
Unary + - ! ~ ++ -- (type) * & sizeof Right to left
Multiplicative * / % Left to right
Additive + - Left to right
Shift << >> Left to right
Relational < <= > >= Left to right
Equality == != Left to right
Bitwise AND & Left to right
Bitwise XOR ^ Left to right
Bitwise OR | Left to right
Logical AND && Left to right
Logical OR || Left to right
Conditional ?: Right to left
Assignment = += -= *= /= %= &= ^= |= <<= >>= Right to left
Comma , Left to right

Example

#include <stdio.h>

int main() {
    int a = 20, b = 10, c = 15, d = 5, e;
    
    // Precedence example 1
    e = a + b * c / d;      // b * c is evaluated first, then divided by d, then added to a
    printf("a + b * c / d = %d\n", e);
    
    // Precedence example 2
    e = (a + b) * c / d;    // a + b is evaluated first due to parentheses
    printf("(a + b) * c / d = %d\n", e);
    
    // Precedence example 3
    e = a + (b * c) / d;    // Same as example 1 but with explicit parentheses
    printf("a + (b * c) / d = %d\n", e);
    
    // Precedence example 4
    e = a + b * (c / d);    // c / d is evaluated first due to parentheses
    printf("a + b * (c / d) = %d\n", e);
    
    // Associativity example
    e = a = b = c;          // Right to left associativity for assignment
    printf("a = b = c results in: a = %d, b = %d, c = %d, e = %d\n", a, b, c, e);
    
    return 0;
}

Output:


a + b * c / d = 50
(a + b) * c / d = 90
a + (b * c) / d = 50
a + b * (c / d) = 50
a = b = c results in: a = 15, b = 15, c = 15, e = 15

Note: When in doubt about operator precedence, use parentheses to make your code more readable and to ensure that operations are performed in the order you intend.

Summary

C provides a rich set of operators for various operations:

  • Arithmetic operators for mathematical calculations
  • Relational operators for comparing values
  • Logical operators for combining conditions
  • Bitwise operators for bit-level operations
  • Assignment operators for assigning values
  • Miscellaneous operators for various purposes

Understanding operator precedence is crucial for writing correct expressions. When in doubt, use parentheses to clarify the order of operations.

Remember that C evaluates expressions from left to right or right to left depending on the operator's associativity, and higher precedence operators are evaluated before lower precedence ones.
