Program Structure

Every C program consists of one or more functions, with main() being the entry point of execution.

#include <stdio.h>

/* This is the main function */
int main() {
    printf("Hello, World!\n");
    return 0;
}

Basic Components of a C Program

  • Preprocessor Directives: Lines starting with # that are processed before compilation.
  • Functions: Blocks of code that perform specific tasks.
  • Statements: Instructions that are executed in sequence.
  • Comments: Notes for human readers that are ignored by the compiler.

The main() function returns an integer value to the operating system. By convention, 0 indicates successful execution.

Compilation Process

Converting C source code into an executable program involves several steps:

  1. Preprocessing: Expands macros and includes header files.
  2. Compilation: Converts C code to assembly language.
  3. Assembly: Converts assembly code to object code.
  4. Linking: Combines object files and libraries into an executable.
# Basic compilation command
gcc -o program_name source_file.c

# Compilation with optimization
gcc -O2 -o program_name source_file.c

# Compilation with debugging information
gcc -g -o program_name source_file.c

Use -Wall flag to enable all compiler warnings, which can help identify potential issues in your code.

Basic Syntax

Statements and Semicolons

In C, statements are terminated with semicolons:

int x = 10;  // Declaration and initialization
printf("Value: %d\n", x);  // Function call
x = x + 5;  // Assignment

Blocks

Blocks are enclosed in curly braces {} and can contain multiple statements:

if (x > 10) {
    printf("x is greater than 10\n");
    x = 10;
}

Identifiers

Rules for naming variables, functions, etc.:

  • Can contain letters, digits, and underscores
  • Must start with a letter or underscore
  • Case-sensitive (count and Count are different)
  • Cannot use C keywords

Avoid using names that begin with an underscore followed by a capital letter or another underscore, as these are reserved for system use.

Keywords

C has a set of reserved keywords that cannot be used as identifiers:

auto break case char const
continue default do double else
enum extern float for goto
if int long register return
short signed sizeof static struct
switch typedef union unsigned void
volatile while
