if-else Statements

The if statement allows you to execute a block of code conditionally based on whether an expression evaluates to true (non-zero) or false (zero).

Basic if Statement

int age = 18;

if (age >= 18) {
    printf("You are an adult.\n");
}

if-else Statement

int score = 75;

if (score >= 60) {
    printf("You passed!\n");
} else {
    printf("You failed.\n");
}

if-else if-else Statement

int score = 85;

if (score >= 90) {
    printf("Grade: A\n");
} else if (score >= 80) {
    printf("Grade: B\n");
} else if (score >= 70) {
    printf("Grade: C\n");
} else if (score >= 60) {
    printf("Grade: D\n");
} else {
    printf("Grade: F\n");
}

Nested if Statements

int age = 25;
int hasLicense = 1;  // 1 for true, 0 for false

if (age >= 18) {
    if (hasLicense) {
        printf("You can drive.\n");
    } else {
        printf("You need a license to drive.\n");
    }
} else {
    printf("You must be at least 18 years old to drive.\n");
}

You can use the ternary operator ? : as a shorthand for simple if-else statements:

int age = 20;
char* status = (age >= 18) ? "adult" : "minor";
printf("You are a %s.\n", status);

switch Statement

The switch statement provides a way to select one of many code blocks to execute based on the value of an expression.

int day = 3;

switch (day) {
    case 1:
        printf("Monday\n");
        break;
    case 2:
        printf("Tuesday\n");
        break;
    case 3:
        printf("Wednesday\n");
        break;
    case 4:
        printf("Thursday\n");
        break;
    case 5:
        printf("Friday\n");
        break;
    case 6:
        printf("Saturday\n");
        break;
    case 7:
        printf("Sunday\n");
        break;
    default:
        printf("Invalid day\n");
}

The break statement is used to exit the switch block. Without it, execution would "fall through" to the next case.

Fall-Through Behavior

char grade = 'B';

switch (grade) {
    case 'A':
        printf("Excellent!\n");
        break;
    case 'B':  // Falls through to case 'C'
    case 'C':
        printf("Well done\n");
        break;
    case 'D':
        printf("You passed\n");
        break;
    case 'F':
        printf("Better try again\n");
        break;
    default:
        printf("Invalid grade\n");
}

The switch statement in C can only test for equality (==), not for other conditions like greater than or less than.

Loops

Loops allow you to execute a block of code repeatedly.

for Loop

The for loop is used when you know in advance how many times you want to execute a block of code.

// Basic for loop
for (int i = 0; i < 5; i++) {
    printf("%d ", i);
}
printf("\n");  // Output: 0 1 2 3 4

// Multiple initialization and increment expressions
for (int i = 0, j = 10; i < j; i++, j--) {
    printf("%d %d\n", i, j);
}

for Loop Components

  1. Initialization: Executed once before the loop starts.
  2. Condition: Evaluated before each iteration. If false, the loop ends.
  3. Increment/Update: Executed after each iteration.

while Loop

The while loop executes a block of code as long as a specified condition is true.

int count = 0;

while (count < 5) {
    printf("%d ", count);
    count++;
}
printf("\n");  // Output: 0 1 2 3 4

do-while Loop

The do-while loop is similar to the while loop, but it executes the block of code once before checking the condition.

int count = 0;

do {
    printf("%d ", count);
    count++;
} while (count < 5);
printf("\n");  // Output: 0 1 2 3 4

The do-while loop always executes at least once, even if the condition is initially false.

Infinite Loops

Loops that run indefinitely until explicitly terminated.

// Infinite for loop
for (;;) {
    // Code to be executed repeatedly
    if (someCondition) break;  // Exit condition
}

// Infinite while loop
while (1) {
    // Code to be executed repeatedly
    if (someCondition) break;  // Exit condition
}

Be cautious with infinite loops. Always ensure there's a way to exit the loop to avoid program hangs.

break and continue Statements

break Statement

The break statement terminates the loop or switch statement and transfers execution to the statement immediately following the loop or switch.

// Using break in a loop
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // Exit the loop when i equals 5
    }
    printf("%d ", i);
}
printf("\n");  // Output: 0 1 2 3 4

continue Statement

The continue statement skips the rest of the current iteration and proceeds with the next iteration of the loop.

// Using continue in a loop
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;  // Skip even numbers
    }
    printf("%d ", i);
}
printf("\n");  // Output: 1 3 5 7 9

goto Statement

The goto statement transfers control to a labeled statement within the same function.

int i = 0;

start:
    if (i < 5) {
        printf("%d ", i);
        i++;
        goto start;
    }
printf("\n");  // Output: 0 1 2 3 4

The goto statement is generally discouraged in modern programming as it can make code harder to understand and maintain. It's better to use structured control flow statements like loops and conditionals.

Common Use Case: Error Handling

One acceptable use of goto is for error handling and cleanup in C:

int process_data() {
    FILE* file = fopen("data.txt", "r");
    if (file == NULL) {
        goto error_handler;
    }
    
    int* buffer = (int*) malloc(100 * sizeof(int));
    if (buffer == NULL) {
        goto cleanup_file;
    }
    
    // Process data...
    
    free(buffer);
    fclose(file);
    return 0;
    
cleanup_file:
    fclose(file);
error_handler:
    return -1;
}
